Git & GitHub Anleitung Navigation

Für Mac

  • ls - zeigt uns die Inhalte eines Verzeichnisses an.
  • cd - navigiert uns in den gewünschten Ordner.
  • mkdir - wir erstellen einen neuen Ordner.
  • cd .. - bringt uns eine Ebene höher im Verzeichnisbaum bzw. zurück.
  • touch - wir erstellen eine neue Datei -> z.B: style.css.
  • clear - löscht den Inhalt des Terminalfensters.
  • rm - löscht eine Datei. Zum Beispiel: rm style.css löscht die Datei style.css.

Für Windows

  • dir - zeigt uns die Inhalte eines Verzeichnisses an.
  • cd - navigiert uns in den gewünschten Ordner.
  • mkdir - wir erstellen einen neuen Ordner.
  • cd .. - bringt uns eine Ebene höher im Verzeichnisbaum bzw. zurück.
  • echo. > Dateiname - wir erstellen eine leere Datei. Hierbei wird der Befehl echo. verwendet, um eine leere Zeile in eine Datei zu schreiben und so eine leere Datei zu erstellen.
  • Command + K - löscht den Inhalt des Terminalfensters.
  • del - löscht eine Datei. Zum Beispiel: del style.css löscht die Datei style.css.
