Git & GitHub Anleitung Navigation

GitHub

Die Vorgehensweise:

  1. Repo erstellen
  2. Ordner/Projekt erstellen und in VS Code öffnen
  3. Terminal öffnen in VS Code (Shortcut: Command J)
    • git init -> Initialisiert ein neues Git-Repository
    • git add . -> Fügt alle Dateien zum Staging-Bereich hinzu (A = Added)
    • git commit -m 'first commit' -> Erstellt einen Commit mit Kommentar
    • it branch -m main -> Benennt den Hauptzweig in "main" um
    • git remote add origin 'url' -> Verknüpft das lokale Repository mit dem Remote-Repository
    • git push -u origin main -> Pusht den lokalen Hauptzweig zum Remote-Repository

    für ein bestehendes Projekt

    • git add . -> Fügt alle Änderungen im Projekt zum Staging-Bereich hinzu
    • git add "assets/styl.css" -> Fügt eine bestimmte Datei zum Staging-Bereich hinzu
    • it commit -m 'Kommentar...' -> Erstellt einen Commit mit Kommentar
    • git push -> Pusht die Änderungen zum Remote-Repository

Ich will mir das Projekt Klonen

  1. Wir erstellen lokal einen Ordner
  2. GitHub -> Repo -> kopieren uns den Link
  3. VS Code Terminal öffnen (Shortcut: Command J)
    • Terminal öffnen "Command J"
    • git clone LINK - Klont das Remote-Repository lokal
  4. Wir müssen uns in den Ordner navigieren:
    1. cd "Projekt Name" - Navigiert in den gerade erstellten Projektordner
    2. Wenn wir ein Projekt mit mehreren Ordnern haben, müssen wir uns mit cd weiter in die Verzeichnisse navigieren
    3. Um wieder aus dem Ordner herauszunavigieren -> cd ..
  5. Nach dem Klonen landet man nicht direkt im Projektordner, sondern im übergeordneten Verzeichnis.

Wenn ihr ein Projekt löschen bzw. ein Repo löschen wollt, WICHTIG! Der git-Ordner muss gelöscht werden -> Command + Shift + .
